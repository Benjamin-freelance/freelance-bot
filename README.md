# Benjamin Freelance Bot

Bot Telegram IA pour automatiser la rédaction de contenu freelance.

## Variables d'environnement à configurer sur Railway

| Variable | Valeur |
|---|---|
| `TELEGRAM_TOKEN` | Ton token BotFather |
| `ANTHROPIC_KEY` | Ta clé API Anthropic |

## Déploiement

1. Push ce repo sur GitHub
2. Connecte Railway à ce repo GitHub
3. Ajoute les variables d'environnement
4. Deploy !

## Utilisation

1. Ouvre ton bot sur Telegram
2. Envoie `/start`
3. Colle ton brief de mission
4. Valide ou modifie le contenu généré
5. Copie et livre au client
